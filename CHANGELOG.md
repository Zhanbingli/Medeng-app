# 更新日志

## [1.0.1] - 2025-10-08

### 🐛 Bug修复

#### Study卡片翻转问题修复
**问题描述**：
- 在Study页面点击"Show Answer"后，卡片使用3D翻转效果（rotation3DEffect）
- 这导致答案面的文字被镜像翻转，显示为反向文字
- 用户无法正常阅读答案内容

**解决方案**：
- 移除3D翻转动画（rotation3DEffect）
- 使用简单的淡入淡出和缩放组合过渡效果（opacity + scale）
- 保持流畅的动画体验，同时确保文字显示正常

**改进内容**：
- ✅ 答案面文字现在可正常阅读
- ✅ 添加了图标标识（Definition, Etymology, Example）
- ✅ 改进了答案面的布局和间距
- ✅ 使用`.fixedSize(horizontal: false, vertical: true)`确保长文本正确换行
- ✅ 更清晰的视觉层次结构

**技术细节**：
```swift
// 之前（有问题）：
.rotation3DEffect(
    .degrees(isShowingAnswer ? 180 : 0),
    axis: (x: 0, y: 1, z: 0)
)

// 现在（已修复）：
.transition(.opacity.combined(with: .scale(scale: 0.95)))
```

### 📱 部署配置

#### iOS版本兼容性调整
**更新**：
- 降低部署目标从 iOS 18.4 到 iOS 17.0
- 现在支持更广泛的iOS设备
- 兼容 iOS 17.0 - iOS 18.4

**原因**：
- 用户设备运行 iOS 18.3.1
- 原始部署目标为 iOS 18.4 导致无法安装
- 降级后可在真机上正常测试
